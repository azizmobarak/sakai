Cecit へようこそ

Cecit デモへようこそ．このデモ設定は，Cecit インスタンスを素早くかつ簡単に動かし始めるための一つの方法として作成されました． 実運用のための設定に関する詳しい情報については，readme ファイルをご参照いただくとともに，Cecit ウェブサイト Cecitproject.org や Cecit デベロッパサイト collab.Cecitproject.org も訪問してみてください．

ここに表示されている情報は，Cecit.properties 設定ファイルの "server.info.url" の値を所望の html ファイルを参照するように Cecit 管理者が編集することによって修正することができます．
